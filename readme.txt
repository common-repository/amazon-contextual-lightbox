== Description ==

Display an Amazon contextual widget inside a lightbox popup. Bring more attention to amazons high quality range of products and make more money!

***Benefits:***

*   Increase clicktrough rate
*   If people land on amazon they will usually buy other stuff too. You get paid for that stuff also 
*   Can be shown once every $x days per visitor so it doesn\'t get annoying  
*   Features an standard search query (user has ability to search too).

***Options***

*   Can be shown  once every defined amount of days
*   Standard Amazon Search Query can be inputed
*   Delay before showing (default value is the best)
*   Disable loading fancybox.js (if the theme allready loads it)

***Future Functionality***

*   Dynamic Standard Search Query (Based on tags)
*   Request: <a href=\"http://arevico.com/contact-us/\">http://arevico.com/contact-us/</a>  
    

***Requirement***

*   Amaon Associate tag (looks like: name-20)
*   PHP 5
*   Valid Doc Type in your theme (fancybox requires this officially, but most themes have a valid doctype allready)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
